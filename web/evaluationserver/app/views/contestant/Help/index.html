Welcome to evaluation server! You're currently logged as contestant. In this role you may participate in several competitions!

On the top of the screen you can see menu. It will change, depends on page you are. In the next paragraphs will be described things you can do.

Competitions

On this page, you see list of competitions you're registred in. There are two types, training and competitions. Competition has exact start and end time. Training is a long-timed competition that shouldn't be time-bounded.

To enter selected competition click on its name. Now, you can see description of the competition. You have several options to do. You may read tasks, look at your statistics, see your solutions, or submit new solution.

Tasks

This page shows list of tasks that you can solve in selected competition. When one is selected, you should see its description. It contains:

  • Name - name of task
  • Time limit - how long will server wait for response of program
  • Memory limit - maximum of allocated memory for program
  • Source limit - maximum size of source code
  • Description - description of problem
  • Sample input - valid source to program
  • Sample output - valid response to given input

After you solve problem, go to page "Submit solution".

Submit solution

When you have written source code of program that you think is correct, you could send it to server to check it. It's neccessery to select right task, programming language and upload source code of your program.

After you upload solution, you can get one of this responses:

  • Accepted - your solution is correct
  • Compile error - uploaded source code can't be correctly compiled
  • Presentation error - your solution seems right, but it's not correctly formated
  • Wrong answer - your solution is wrong
  • Runtime error - some kind of exception happends during execution
  • Time limit exceeded - your program is too slow
  • Memory limit exceeded - your program asked for more memory that is limit
  • Output limit exceeded - too much data are generated by your program
  • Restricted function - program tried to use forbidden function
  • Internal error - something bad happens, contact administrator

Solutions

There is a list of your solutions. You can see time when you send it, used memory, time taken to sove it. You could also download solution. Main column is reply, that show correctness of the soulution.

Statistics

If you want to see your current progress, statistics is the right page. There are 2 kind of statistics. First shows you tasks statistics. Second your position in competition. The second statistics will stop refreshing before the competition ends. Exact time depends on setting of competition.
